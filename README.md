

## Keen Keeper - Keep Your Friendships Alive
---
### Description
- Keen Keeper is a modern web application that helps users to track and maintain meaningful connections with their friends. It provides a user-friendly interface for storing and organizing friends details, as well as tracking friendship activities. In the homepage, the website displays a summary card section that provides a quick overview of key metrics, such as total friends, on-track friends, and friends needing attention. It also features a friends section where users can view all their friends. Clicking on a friend card user can view their details.
Users can call, text, or video chat with their friends, and the web app will automatically record the dates of these interactions. In the timeline page, users can view those friendship interactions. Lastly, the stats page provides a summary of the number of calls, texts, and video interactions with a pie chart. The app focuses on simplicity, responsiveness, and a smooth user experience using Next.js.
---
### Technologies Used
- Next.js (App Router)
- React
- Tailwind CSS + DaisyUI
- Recharts
- React Toastify
- Context API
- React Icons
---
### Features
- Track and filter Interactions (Calls, Texts, Video) in a clean, cronological timeline with date and visual indicators. Interactions are stored using local storage, ensuring persistence across sessions.
- Analyze interactions using a pie chart of the number of calls, texts, and video interactions.
- See individual friend details by clicking on a friend card and interacting with them.
